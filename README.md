# Complyer-SAT

A simple SAT solver which uses Tseytin Transformation for conversion of expressions to CNF and uses the following algorithms to find a satisfying assignment for the expression

- 2-SAT: _O(N)_
- HORN-SAT: _O(N)_
- DPLL: _O(2<sup>N</sup>)_
